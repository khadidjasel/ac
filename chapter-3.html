Chapitre 3 : Le modèle OSI

A) Les Composants matériel d’un Réseau informatique :

  1. Carte réseau : « d’interface réseau », est un composant essentiel pour se connecter à un réseau informatique et de communiquer avec d’autres ordinateurs sur le même réseau, ainsi que d’accéder à Internet.
  2. Switch : est un dispositif qui permet de connecter plusieurs appareils sur un réseau informatique local. Il permet une communication efficace en envoyant les données uniquement vers le destinataire approprié utilisant des adresses MAC et peut offrir des fonctionnalités avancées pour la gestion et la surveillance du réseau.
  3. Connecteur HUB : est un appareil qui permet de connecter plusieurs appareils sur un réseau informatique. Le HUB sert de point central pour la communication entre les différents appareils. Mais il est aujourd’hui largement remplacé par les commutateurs réseau.

B) Le Modèle OSI :

Le modèle OSI a été conçu pour permettre l’interopérabilité entre différents systèmes informatiques en standardisant les protocoles et les interfaces de communication, mais il a été largement remplacé par le modèle TCP/IP dans les réseaux modernes. Il définit une structure en sept couches, chacune ayant une fonction spécifique. Ces couches sont :

  1. La couche physique : responsable de la transmission des données sur le support physique (par exemple, câble, fibre optique, ondes radio).
  2. La couche liaison de données : responsable de la transmission des données entre les appareils qui partagent un même support physique.
  3. La couche réseau : responsable du routage des données à travers le réseau.
  4. La couche transport : responsable de l’acheminement des données de bout en bout entre les appareils.
  5. La couche session : responsable de l’établissement et de la gestion des sessions entre les appareils.
  6. La couche présentation : responsable de la conversion des données dans un format qui peut être compris par l’appareil de destination.
  7. La couche application : responsable de la communication entre les applications sur les appareils de réseau.

Le modèle OSI a été conçu pour permettre l’interopérabilité entre différents systèmes informatiques en standardisant les protocoles et les interfaces de communication, mais il a été largement remplacé par le modèle TCP/IP dans les réseaux modernes.

C) Le Modèle TCP/IP :

Le modèle TCP/IP est un modèle de référence pour les communications de réseau informatique. Il est utilisé pour l’interconnexion des réseaux informatiques à travers le monde. Le nom TCP/IP vient des deux protocoles principaux qui le composent : le protocole de contrôle de transmission (TCP) et le protocole Internet (IP). Le modèle TCP/IP est divisé en quatre couches : la couche application, la couche transport, la couche Internet et la couche réseau d'accès. Chaque couche est responsable d'une fonction spécifique dans la communication de données entre les ordinateurs. La couche application est responsable des protocoles de haut niveau tels que HTTP, FTP et SMTP. La couche transport est responsable de la transmission de données entre les applications utilisant TCP ou UDP. La couche Internet est responsable du routage des paquets de données à travers les réseaux interconnectés. Enfin, la couche réseau d'accès est responsable de la transmission de données entre les nœuds du réseau local. Le modèle TCP/IP est largement utilisé dans le monde entier pour la communication entre les ordinateurs et les réseaux informatiques. Il fournit une structure logique pour la communication de données et facilite l'interconnexion des réseaux hétérogènes.
