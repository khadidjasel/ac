Introduction

Le web et internet c’est pareil :

  • Le WWW : la « toile mondiale ». Système hypertexte public sur Internet. Le Web permet de consulter, avec un navigateur, des pages accessibles sur des sites.
  • Principe World Wide Web : Accéder à des documents textuels qui sont situés sur des machines accessibles par Internet et reliés entre eux par un mécanisme de lien « hypertexte »

Un lien hypertexte : est un lien qui permet d’ouvrir une nouvelle page (URL).

URL : décrivant l’emplacement d’une ressource. Elle contient l’adresse du serveur, et l’adresse de la page sur le serveur.

Aspects techniques du Web

Les 3 mécanismes de base du Web sont :

  1. URL : Le mécanisme de localisation peut faire appel au protocole DNS.
  2. HTTP : Protocole de niveau applicatif et sans état.
  3. HTML : Langage de description de « pages Web » avec texte, images et autres objets.

Page web et site web

Une page Web, c’est un document multimédia : images, textes, possibilité d’interaction, liens qui décrit élément par élément : titre, morceaux de texte, images. Stocké dans un fichier html.

Création d’une page web HTML

Un document HTML est un Langage de description de pages web et aussi un fichier texte, construit suivant une syntaxe précise qui est structuré par des balises interprétées dans un navigateur.

Les principes de langage HTML :

  • <html> … </html> : Délimite le début et la fin du document
  • <head> … </head> : Entête du document
  • <title> … </title> : Titre du document
  • <body> … </body> : Corps du document

Les titres et paragraphes :

  • <hn> … <hn> : Titre de niveau n :1 à 6
  • <p> … <p> : paragraphe
